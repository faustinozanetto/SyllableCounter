# Syllable Counter Test

This project is a simple Node.js application for counting syllables in words using a CSV file. It includes a syllable counting function and a script to process a CSV file with words and correct syllable counts, identifying incorrect ones and saving the results to a file.

## Prerequisites

* Node.js installed on your machine.
* npm (Node.js package manager) installed.

## Installation

1. Clone the repository to your local machine:

```bash
git clone https://github.com/faustinozanetto/SyllableCounter
```

2. Navigate to the project directory:

```bash
cd SyllableCounter
```

3. Install the dependencies:

```bash
npm install
```

## Usage

To test the syllable counter simply run the following command:

```bash
npm test
```
